# Pollyush

Polly is a poll app built with React.

  - Create a poll
  - See results
  - Share
  - Analyze with charts



### Tech

* [ReactJS] - HTML enhanced for web apps!
* 


### Installation

 requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd 
$ npm install -d
$ node app
```
